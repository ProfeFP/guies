📘 Manual de Superviviència Digital

Aprèn modificant el codi en temps real.

Bloc I: L'Estructura "Trinitat" (HTML/CSS/JS)

Una pàgina web professional sempre té tres parts: l'esquelet (HTML), el vestit (CSS) i el cervell (JS). En aquest bloc veuràs com conviuen tots tres dins d'un sol fitxer.

Bloc I: Estructura Completa 
    <!DOCTYPE html>
    <html>
    <head>
        <title>La meva primera web</title>
        <style>
            /* AQUÍ VA EL CSS (EL VESTIT) */
            h1 { color: #e67e22; text-align: center; font-family: sans-serif; }
        </style>
        <script>
        // AQUÍ VA EL JAVASCRIPT (EL CERVELL)
        console.log("Hola! La pàgina s'ha carregat correctament.");
    </script>
    </head>
    <body>
    
    <h1>Soc una pàgina HTML</h1>
           
    </body>
    </html>

Bloc II: La Configuració "Pro" (Metadades, UTF-8 i Bootstrap)

Dins del <head> preparem la web per al món real: accents que es veuen bé, connexió a llibreries externes (Bootstrap) i com es veurà el link quan el compartim per WhatsApp.

Exemple: Estructura Completa del Head 
    <!DOCTYPE html>
    <html lang="ca">
    <head>
        <!-- 1. Codificació per a accents i ç -->
        <meta charset="UTF-8">
    
        <!-- 2. Responsive: Adaptació a mòbils -->
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
        <!-- 3. Metadades OpenGraph (per a Xarxes Socials) -->
        <meta property="og:title" content="Portfoli Docent de Reus">
        <meta property="og:image" content="https://via.placeholder.com/1200x630">
    
        <!-- 4. Connexió a BOOTSTRAP 5 (CSS) -->
        <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
    
        <!-- 5. Tipografia de Google Fonts -->
        <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap" rel="stylesheet">
    
        <style>
            body { font-family: 'Inter', sans-serif; }
            .prova-lletra { font-weight: 700; color: #0d6efd; }
        </style>
    </head>
    <body class="container mt-4">
    
        <h1 class="text-primary">L'educació a l'era digital</h1>
        <p>Gràcies a UTF-8, els accents es veuen perfectes.</p>
        
        <span class="prova-lletra">Soc un text amb la lletra 'Inter' de Google</span>
        
        <!-- Botó amb estil de Bootstrap -->
        <button class="btn btn-success d-block mt-3">Guardar Canvis</button>
    
        <!-- 6. JavaScript de Bootstrap (al final del body) -->
        <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
    </body>
    </html>

Bloc III: Anatomia de l'Etiqueta (Atributs, Classes i ID)

Les etiquetes no només diuen "què són", sinó "com es comporten". Fem servir Atributs per donar informació extra, Classes per agrupar estils i ID per identificar un element únic.

Exemple: Etiquetes amb Superpoders 
        <!-- 1. DIV: El contenidor (Bloc) -->
        <div id="seccio-principal" class="caixa-estil">
            
            <!-- 2. IMG: Imatge amb atributs (src, alt, width) -->
            <img src="https://via.placeholder.com/150" alt="Imatge d'exemple" width="100">
        
            <!-- 3. BR: Salt de línia (sense tancament) -->
            <br>
        
            <!-- 4. A: Enllaç amb atribut ID per a JS -->
            <a href="https://agora.xtec.cat" id="enllaç-escola">Visita la nostra web</a>
        
            <!-- 5. BUTTON: Interacció amb classe -->
            <button class="boto-accions" onclick="alert('Hola!')">Clica'm</button>
        
            <!-- 6. INPUT: L'etiqueta que no podem oblidar per rebre dades -->
            <input type="text" placeholder="Escriu el teu nom..." id="entrada-usuari">
        
        </div>
        
        <style>
            /* Fem servir la CLASSE (.) per estils compartits */
            .caixa-estil { border: 2px solid #3498db; padding: 20px; border-radius: 10px; }
            .boto-accions { background: #e74c3c; color: white; border: none; padding: 10px; }
        
            /* Fem servir l'ID (#) per estils ÚNICS */
            #enllaç-escola { font-weight: bold; color: green; display: block; margin: 10px 0; }
        </style>

Bloc IV: El Disseny en Detall (Estils de Text i Imatge)

El CSS ens permet controlar cada detall visual. Aquí veurem com transformar un text simple en un titular elegant i com fer que una imatge sembli professional amb vores i ombres.

Exemple: Tipografia i Maquillatge Visual 
      <!-- 1. Text amb estil treballat -->
      <h2 class="titol-elegant">Benvinguts a l'Aula Digital</h2>
      <p class="paragraf-modern">
          Aquest text té un interlineat i un espaiat pensats per a una lectura còmoda.
      </p>
      
      <!-- 2. Imatge amb estil professional -->
      <img src="https://picsum.photos/300/200" class="imatge-estilitzada" alt="Foto de mostra">
      
      <style>
          /* Estils per al TEXT */
          .titol-elegant {
              color: #2c3e50;
              font-family: 'Segoe UI', sans-serif;
              font-size: 2.5rem;
              text-shadow: 2px 2px 4px rgba(0,0,0,0.1);
              border-bottom: 3px solid #3498db;
              display: inline-block;
              margin-bottom: 15px;
          }
      
          .paragraf-modern {
              color: #7f8c8d;
              line-height: 1.8; /* Espai entre línies */
              letter-spacing: 0.5px; /* Espai entre lletres */
              text-align: justify;
          }
      
          /* Estils per a la IMATGE */
          .imatge-estilitzada {
              width: 100%;
              max-width: 300px;
              border-radius: 15px; /* Vores arrodonides */
              box-shadow: 0 10px 20px rgba(0,0,0,0.2); /* Ombra profunda */
              border: 5px solid white;
              transition: transform 0.3s ease; /* Animació suau */
          }
      
          .imatge-estilitzada:hover {
              transform: scale(1.05); /* S'engrandeix en passar el ratolí */
          }
      </style>

Bloc V: L'Ordre Lògic (Semàntica HTML5)

No totes les caixes són iguals. L'HTML5 ens dóna etiquetes amb significat perquè els cercadors i les eines d'accessibilitat entenguin quina part és el menú, quina el contingut i quina el peu de pàgina.

Exemple: Estructura d'una Pàgina Real 
    <!-- El cap de la web (Logo, títol, menú) -->
    <header>
        <h1>Institut de Reus</h1>
        <nav>
            <ul>
                <li><a href="#">Inici</a></li>
                <li><a href="#">Projectes</a></li>
            </ul>
        </nav>
    </header>
    
    <!-- El contingut principal i únic de la pàgina -->
    <main>
        <section id="noticies">
            <h2>Últimes Notícies</h2>
            <article>
                <h3>Guanyem el premi d'Innovació</h3>
                <p>L'alumnat de 4t d'ESO ha creat una web...</p>
            </article>
        </section>
    
        <aside>
            <p>Barra lateral: Enllaços d'interès o avisos ràpids.</p>
        </aside>
    </main>
    
    <!-- El final de la web (Contacte, xarxes, copyright) -->
    <footer>
        <p>© 2026 - Creat amb finalitats docents</p>
    </footer>
    
    <style>
        header { background: #34495e; color: white; padding: 10px; }
        main { display: flex; gap: 20px; padding: 20px; }
        section { flex: 3; }
        aside { flex: 1; background: #ecf0f1; padding: 10px; }
        footer { text-align: center; background: #2c3e50; color: white; padding: 10px; }
    </style>

Bloc VI: L'Art d'endreçar l'espai (Flexbox)

Com fem que el menú estigui a l'esquerra i el contingut a la dreta? Amb display: flex, el contenidor es torna "elàstic" i ordena els seus elements de manera automàtica.

Exemple: Graella de Recursos 
    <!-- El contenidor PARE defineix les regles -->
    <div class="zona-flex">
        <div class="targeta">🍎 Recurs A</div>
        <div class="targeta">📚 Recurs B</div>
        <div class="targeta">💻 Recurs C</div>
    </div>
    
    <style>
        .zona-flex {
            display: flex;            /* Activem el mode flexible */
            justify-content: space-around; /* Reparteix l'espai entre elements */
            align-items: center;      /* Centra verticalment */
            flex-wrap: wrap;          /* Si no caben, salten a la línia següent (Mòbil!) */
            gap: 20px;               /* Distància de seguretat entre caixes */
            background: #f8f9fa;
            padding: 30px;
            border: 1px dashed #ccc;
        }
    
        .targeta {
            background: white;
            border-bottom: 4px solid #0d6efd;
            padding: 20px;
            flex: 1;                 /* Totes intenten mesurar el mateix */
            min-width: 200px;        /* Però no es fan més petites que 200px */
            text-align: center;
            box-shadow: 0 4px 8px rgba(0,0,0,0.1);
        }
    </style>

Bloc VII: El Cervell de la Web (Validació amb JavaScript)

JavaScript ens permet comprovar les dades abans d'enviar-les. En aquest exemple, el botó d'enviar estarà bloquejat (desactivat) fins que l'usuari escrigui un correu electrònic amb el format correcte.

Exemple: Formulari Intel·ligent 
    <!-- Formulari simple de Bootstrap -->
    <div class="p-4 border rounded bg-light">
        <div class="mb-3">
            <label for="emailInput" class="form-label">Correu electrònic d'escola:</label>
            <input type="email" class="form-control" id="emailInput" placeholder="nom@escola.cat">
            <div id="avis-error" class="text-danger mt-1" style="display:none;">L'email no és vàlid encara...</div>
        </div>
        
        <button id="botoEnviar" class="btn btn-primary" disabled>Enviar dades</button>
    </div>
    
    <script>
        const inputEmail = document.getElementById('emailInput');
        const boto = document.getElementById('botoEnviar');
        const avis = document.getElementById('avis-error');
    
        // Escoltador d'esdeveniments: "quan l'usuari escrigui..."
        inputEmail.addEventListener('input', function() {
            const valor = inputEmail.value;
            
            // Regla de validació simple (conté @ i .)
            const esValid = valor.includes('@') && valor.includes('.');
    
            if (esValid) {
                boto.disabled = false; // Activem el botó
                avis.style.display = 'none';
                inputEmail.classList.add('is-valid');
                inputEmail.classList.remove('is-invalid');
            } else {
                boto.disabled = true; // Bloquegem el botó
                avis.style.display = 'block';
                inputEmail.classList.add('is-invalid');
                inputEmail.classList.remove('is-valid');
            }
        });
    </script>

Bloc VIII: Connexió Real (Notícies del Diari ARA)

La web pot "llegir" altres mitjans. Aquí connectem amb l'RSS de l'Ara.cat, processem les dades i les maquetem amb imatges i descripcions en temps real.

Exemple: Noticiari Dinàmic 
    <div class="container mt-4">
        <h3 class="mb-4 text-danger border-bottom pb-2">Últimes notícies: Diari ARA</h3>
        <div id="noticiari" class="row g-4">
            <p class="text-muted">Connectant amb el diari...</p>
        </div>
    </div>
    
    <script>
        async function carregarAra() {
            const contenidor = document.getElementById('noticiari');
            const rssUrl = 'https://www.ara.cat/rss/latest/';
            // Fem servir un convertidor RSS a JSON gratuït per evitar bloquejos (CORS)
            const apiProxy = `https://api.rss2json.com/v1/api.json?rss_url=${encodeURIComponent(rssUrl)}`;
    
            try {
                const resposta = await fetch(apiProxy);
                const dades = await resposta.json();
    
                contenidor.innerHTML = ""; // Netegem el missatge de càrrega
    
                // Agafem les primeres 3 notícies
                dades.items.slice(0, 3).forEach(item => {
                    const col = document.createElement('div');
                    col.className = "col-md-4";
                    
                    // Busquem la imatge (en el format JSON sol venir a 'enclosure' o 'thumbnail')
                    const imatge = item.enclosure.link || item.thumbnail || 'https://via.placeholder.com/300x200?text=ARA';
    
                    col.innerHTML = `
                        <div class="card h-100 shadow-sm">
                            <img src="${imatge}" class="card-img-top" style="height:180px; object-fit:cover;" alt="Foto">
                            <div class="card-body">
                                <h5 class="card-title">${item.title}</h5>
                                <p class="card-text text-muted small">${item.description.substring(0, 100)}...</p>
                                <a href="${item.link}" target="_blank" class="btn btn-outline-danger btn-sm">Llegir més</a>
                            </div>
                        </div>
                    `;
                    contenidor.appendChild(col);
                });
            } catch (error) {
                contenidor.innerHTML = "<p class='text-danger'>No s'han pogut carregar les notícies.</p>";
            }
        }
    
        carregarAra();
    </script>
