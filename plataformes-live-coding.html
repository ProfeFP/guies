Plataformes de Live Coding

Escull l'entorn perfecte per programar al navegador i aprèn els 30 conceptes clau del desenvolupament web modern

💡

Les plataformes de live coding permeten escriure i veure el resultat del codi en temps real, sense instal·lar res. Són ideals per a tallers, exercicis en classe i per compartir codi fàcilment. A continuació trobaràs una comparativa i, a sota, el vocabulari tècnic essencial per fer prompts de qualitat a una IA.

Secció 01

Comparativa de Plataformes

Plataforma Dificultat Aspecte Visual Col·laboració Recomanació
CodePen ↗ codepen.io
Molt Baixa ★★★★★ Molt fàcil (Fork) ⭐ Millor per tallers
StackBlitz ↗ stackblitz.com
Mitjana ★★★★★ Excel·lent Per a docents avançats
PlayCode ↗ playcode.io
Baixa ★★★★★ Bàsica Ràpida per a proves
CodeSandbox ↗ codesandbox.io
Mitjana/Alta ★★★★★ Molt bona (multi-usuari) Potent, complexa al inici
Liveweave ↗ liveweave.com
Baixa ★★★★★ Bona Eines auxiliars visuals

🖊 CodePen

codepen.io

La millor opció per a principiants. Interfície molt neta dividida en 3 panells (HTML / CSS / JS) i un resultat en directe. La funció Fork permet copiar qualsevol exercici i modificar-lo.

Molt Baixa ⭐ Ideal per al taller

⚡ StackBlitz

stackblitz.com

Experiència de desenvolupador real. És pràcticament VS Code al navegador, amb terminals i estructura de fitxers. Molt ràpid de carregar i ideal per als docents que ja saben una mica.

Mitjana Per a perfils avançats

▶ PlayCode

playcode.io

Simplicitat al màxim. Molt ràpid per provar idees sense ni registrar-se. Ideal per a demostracions de 5 minuts o exercicis flash sense complicacions.

Baixa Proves ràpides

📦 CodeSandbox

codesandbox.io

La plataforma de col·laboració en viu: diversos alumnes poden editar el mateix fitxer simultàniament, com un Google Docs del codi. Molt potent però pot marejar al principi.

Mitjana/Alta Col·laboració multi-usuari

🧵 Liveweave

liveweave.com

Pensada per al disseny. Inclou regles visuals, selector de colors i guies integrades a la mateixa pantalla. Perfecta quan l'objectiu és aprendre CSS de forma visual.

Baixa Eines de disseny visuals
Secció 02

Diccionari de Conceptes per al Prompt Engineering

🗝️

Per fer prompts de qualitat a una IA, cal conèixer el vocabulari tècnic. Si no saps que existeix el concepte Responsive, mai no el demanaràs. Aquí tens els 30 conceptes clau organitzats per capes — el teu vocabulari d'arquitecte de prompts.

I

Fonaments i Estructura (HTML5)

7 conceptes · L'esquelet de la web
▼
📄
Doctype HTML
Declara que usem HTML5 al navegador
🌳
<html> Root
Contenidor pare de tot el document
✂️
Head vs Body
Configuració invisible vs contingut visual
🏷️
Semàntica HTML5
<header>, <main>, <footer>, <section>, <article>
🔬
Anatomia Etiqueta
Obertura, contingut i tancament
⚙️
Atributs
href, src, alt, id, class...
🔗
Hipervincles i Rutes
Relatives (internes) vs absolutes (URL externes)
II

Configuració Invisible (Metadades)

5 conceptes · El que el navegador llegeix però l'usuari no veu
▼
🔤
Charset UTF-8
Permet accents i caràcters especials
📱
Viewport Meta
Evita que la web es vegi "en miniatura" al mòbil
🔍
Metadades SEO
description i keywords per als cercadors
📣
Open Graph (og:)
Com es veu la web a WhatsApp o xarxes socials
🌐
Favicon
La petita icona de la pestanya del navegador
III

Disseny i Estètica (CSS)

9 conceptes · El vestit de la web
▼
🎯
Selectors CSS
Per etiqueta, classe (.) o ID (#)
📦
Box Model
margin, border, padding i content
📏
Unitats de Mesura
px (fixos) vs rem, %, vh (relatius)
↔️
Flexbox
Alinear elements en eix principal i secundari
⬛
CSS Grid
Dissenys de graella complexos en 2D
📲
Responsive Design
Media Queries per canviar disseny per pantalla
🔡
Google Fonts
Importar i aplicar tipografies externes
✨
Efectes Visuals
box-shadow, border-radius, opacity
🎞️
Transicions i :hover
Pas suau entre estats interactius
IV

Biblioteques i Frameworks

2 conceptes · Eines predefinides per guanyar velocitat
▼
🧰
Frameworks CSS
Bootstrap, Tailwind — classes predefinides per disseny
🌍
CDN
Carregar llibreries sense descarregar fitxers locals
V

Interactivitat i Lògica (JavaScript)

5 conceptes · El cervell de la web
▼
🗺️
DOM
El "mapa" que permet al JS modificar l'HTML en viu
📦
Variables i Tipus
strings, numbers, booleans — on guardem info
👆
EventListeners
Clics, teclejos, scroll, moviments de ratolí
🔧
Funcions
Blocs de codi reutilitzables per a una tasca
🔀
Condicionals if/else
La lògica de presa de decisions de la web
VI

Disseny Flexible i Adaptabilitat

4 conceptes · La web que "reacciona" a l'entorn
▼
📐
Media Queries
Estils CSS que canvien segons la mida de la pantalla
✂️
Breakpoints
Punts de tall en píxels on el disseny canvia
📲
Mobile-First
Dissenyar primer per mòbil, després per desktop
📏
vh, vw, %
Mides proporcionals a la pantalla, no fixes
VII

Lògica i Validació (JS Avançat)

4 conceptes · La web com a aplicació
▼
🗺️
DOM (avançat)
Canviar textos, amagar botons, modificar colors en temps real
🎛️
Events: submit, scroll
Més que el clic: formularis, desplaçament, hover
🔎
RegEx (Validació)
Comprovar si un text té forma d'email o DNI
💾
Variables i États
La web "recorda" si l'usuari ha iniciat sessió, etc.
VIII

Connexió i Dades Externes (APIs, RSS & JSON)

6 conceptes · La web connectada al món
▼
🍽️
API
El "cambrer" que busca dades al servidor per a l'usuari
📡
RSS
Format robust per subscriure's a notícies i blogs
📋
JSON
L'idioma universal de les dades a Internet
⏳
Asincronia / Fetch
Carregar dades en segon pla sense recarregar la pàgina
🧱
Frameworks JS
React, Vue, jQuery — faciliten la connexió de dades
🔒
CORS
Per què el navegador bloqueja recursos d'altres webs

Del Prompt Bàsic al Súper-Prompt

🚀 Com evoluciona el teu prompt quan coneixes el vocabulari

❌ Prompt sense vocabulari tècnic
"Fes-me una web de notícies"
✅ Prompt d'Arquitecte amb els 30 conceptes
"Genera una pàgina HTML5 semàntica amb un header fix i un main configurat amb Flexbox. Inclou metadades Open Graph i configuració UTF-8. Connecta la llibreria Bootstrap 5 via CDN per al disseny de les targetes (cards) de notícies i usa la Fetch API per llegir un JSON extern asíncronament. Aplica una tipografia Google Font tipus 'Inter' al body i assegura't que sigui Responsive amb un breakpoint per a mòbil."
