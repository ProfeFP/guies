Per què Google Colab?

Google Colab és molt més que un editor de codi. És un quadern interactiu (Jupyter Notebook) allotjat al núvol que barreja text explicatiu, codi executable i resultats en un sol document. Per a l'aula, això canvia completament la manera d'aprendre a programar: el professor pot compartir el quadern amb tota la classe, els alumnes hi executen el codi pas a pas, veuen els resultats immediatament i l'entreguen amb els exercicis resolts al mateix fitxer.

☁️

Sense instal·lació

Funciona al navegador. Python, llibreries i entorn ja configurat. Cap problema amb versions ni sistemes operatius.

🤝

Treball col·laboratiu

Com un Google Docs però per a codi. El professor pot veure el treball dels alumnes en temps real i deixar comentaris.

📦

Cel·les independents

Cada cel·la s'executa per separat. Pots provar un fragment, corregir-lo i tornar-lo a executar sense reiniciar tot.

📝

Text + codi barrejat

Les cel·les de text (Markdown) permeten afegir explicacions, títols i notes directament al costat del codi.

🖥️

GPU gratuïta

Per a projectes avançats, Colab ofereix accés gratuït a GPU, ideal per a Machine Learning i processament de dades.

💾

Integrat amb Drive

Es guarda automàticament a Google Drive. Accessible des de qualsevol dispositiu amb el compte de Google.

Com funciona un quadern Colab

Un quadern es divideix en cel·les. Les cel·les de codi s'executen amb el botó ▶ o prement Ctrl+Enter. Les cel·les de text contenen explicacions en format Markdown. Pots afegir noves cel·les, reordenar-les i compartir el quadern complet amb un sol enllaç.

1. Variables i Tipus de Dades

Una variable és un contenidor que guarda un valor a la memòria. A Python no cal declarar el tipus: s'infereix automàticament. Cada cel·la de Colab pot definir variables que estaran disponibles per a les cel·les posteriors.

🔢

int / float

Nombres enters i decimals. edat = 17 o nota = 8.5

📝

str

Cadenes de text entre cometes. nom = "Anna"

☑️

bool

Valors lògics. aprobat = True o False

🔍

type()

Consulta el tipus. type(edat) retorna <class 'int'>

📓 Cel·la Colab codi 
# ── 1. VARIABLES ─────────────────────────────

# Tipus numèrics
edat   = 17
nota   = 8.5
nom    = "Anna"
aprobat = True

# Mostrar valors i tipus
print(f"Nom: {nom} | Edat: {edat} anys")
print(f"Nota: {nota} | Aprovat: {aprobat}")
print(f"Tipus de nota: {type(nota)}")

# Conversió de tipus
nota_text = str(nota)   # float → string
edat_real = float(edat) # int → float
▶ Sortida:
Nom: Anna | Edat: 17 anys
Nota: 8.5 | Aprovat: True
Tipus de nota: <class 'float'>
🎓 Proposta d'exercici Colab

Afegeix una nova cel·la al quadern i defineix 4 variables pròpies: el teu nom, l'any de naixement, la nota mitjana i si t'agrada la programació. Mostra-les amb un print() formatat. Executa la cel·la i comprova que el resultat és correcte.

2. Condicionals (if / elif / else)

Els condicionals permeten que el programa prengui decisions. A Python l'indentació (4 espais) és obligatòria i defineix els blocs de codi. A Colab pots provar ràpidament condicions canviant el valor de les variables a la cel·la anterior i tornant a executar.

📓 Cel·la Colab codi 
# ── 2. CONDICIONALS ──────────────────────────

nota = 6.8

if nota >= 9:
    print("Excel·lent 🌟")
elif nota >= 7:
    print("Notable 👍")
elif nota >= 5:
    print("Aprovat ✅")
else:
    print("Suspès ❌ — Cal repassar!")

# Operadors lògics: and, or, not
edat = 17
te_carnet = False

if edat >= 18 and te_carnet:
    print("Pot conduir")
else:
    print("No pot conduir")
▶ Sortida:
Aprovat ✅
No pot conduir
🎓 Proposta d'exercici Colab

Modifica el valor de nota a 3, 5, 7.5 i 9.2 i executa la cel·la cada vegada. Observa com canvia la sortida. Afegeix un elif per a la condició "Matrícula d'honor" per a notes ≥ 9.5.

3. Bucles (for / while)

Els bucles repeteixen un bloc de codi múltiples vegades. for itera sobre una seqüència coneguda; while repeteix mentre es compleixi una condició. A Colab és molt útil per processar llistes de dades o generar seqüències.

📓 Cel·la Colab codi 
# ── 3. BUCLES ────────────────────────────────

# for amb range()
print("Taula del 3:")
for i in range(1, 11):
    print(f"  3 × {i} = {3 * i}")

# for sobre una llista
alumnes = ["Anna", "Bernat", "Carla", "David"]
for alumne in alumnes:
    print(f"👤 {alumne}")

# while amb comptador
contador = 0
while contador < 5:
    print(f"Volta {contador + 1}")
    contador += 1

print("Bucle acabat!")
▶ Sortida:
Taula del 3:
  3 × 1 = 3
  3 × 2 = 6
  ...
👤 Anna
👤 Bernat
...
Volta 1 · Volta 2 · ... · Volta 5
Bucle acabat!
🎓 Proposta d'exercici Colab
  • Crea una llista amb els teus 5 videosjocs / pel·lícules / cançons preferides i recorre-la amb un for mostrant cada element numerat.
  • Afegeix un while que acumuli la suma dels números de l'1 al 100 i mostri el resultat.

4. Llistes i Diccionaris

Les llistes guarden col·leccions ordenades d'elements. Els diccionaris associen claus amb valors, com una base de dades senzilla. Tots dos s'utilitzen constantment en programació real.

📓 Cel·la Colab codi 
# ── 4. LLISTES I DICCIONARIS ──────────────────

# Llista — ordenada, modificable
notes = [7.5, 8.0, 6.5, 9.0, 5.5]
print(f"Notes: {notes}")
print(f"Màxima: {max(notes)} | Mínima: {min(notes)}")
print(f"Mitjana: {sum(notes) / len(notes):.2f}")

notes.append(7.0)   # Afegir
notes.sort()        # Ordenar

# Diccionari — clau: valor
alumne = {
    "nom":   "Anna",
    "edat":  17,
    "curs":  "1r DAM",
    "notes": [8, 7, 9]
}

print(f"{alumne['nom']} cursa {alumne['curs']}")
for clau, valor in alumne.items():
    print(f"  {clau}: {valor}")
▶ Sortida:
Notes: [7.5, 8.0, 6.5, 9.0, 5.5]
Màxima: 9.0 | Mínima: 5.5
Mitjana: 7.30
Anna cursa 1r DAM
  nom: Anna · edat: 17 · curs: 1r DAM ...

5. Funcions

Les funcions agrupen codi que es repeteix i li donen un nom. Permeten reutilitzar lògica, llegir millor el codi i treballar en equip: cadascú pot implementar una funció diferent al quadern Colab compartit.

📓 Cel·la Colab codi 
# ── 5. FUNCIONS ──────────────────────────────

# Funció bàsica amb paràmetres i retorn
def calcular_mitjana(notes):
    """Calcula la mitjana d'una llista de notes."""
    if len(notes) == 0:
        return 0
    return sum(notes) / len(notes)

def qualificacio(nota):
    """Retorna la qualificació textual d'una nota."""
    if   nota >= 9:   return "Excel·lent"
    elif nota >= 7:   return "Notable"
    elif nota >= 5:   return "Aprovat"
    else:              return "Suspès"

# Crida a les funcions
notas_classe = [7.5, 8.0, 6.5, 9.0, 5.5]
mit = calcular_mitjana(notas_classe)

print(f"Mitjana: {mit:.2f} → {qualificacio(mit)}")

# Funció amb valors per defecte
def saludar(nom, idioma="ca"):
    salutacions = {"ca": "Hola", "es": "Hola", "en": "Hello"}
    return f"{salutacions.get(idioma, 'Hi')}, {nom}!"

print(saludar("Anna"))
print(saludar("Anna", "en"))
▶ Sortida:
Mitjana: 7.30 → Notable
Hola, Anna!
Hello, Anna!
🎓 Proposta d'exercici col·laboratiu Colab
  • Cada alumne defineix una funció diferent al quadern compartit: es_parell(n), factorial(n), invertir_text(s), comptar_vocals(s)…
  • Al final, tots utilitzen les funcions dels companys en un programa que les combina.
  • El professor revisa el quadern en temps real i deixa comentaris a les cel·les incorrectes.

Flux de Treball a l'Aula

Proposta de dinàmica per treballar amb Google Colab de manera col·laborativa entre professor i alumnes.

1

El professor prepara el quadern

Crea el fitxer .ipynb amb cel·les de text explicatives, exemples resolts i cel·les buides per als exercicis dels alumnes.

2

Compartir l'enllaç

Comparteix el quadern via Google Drive amb permisos de «Qualsevol amb l'enllaç pot editar» o crea còpies individuals per alumne via Classroom.

3

Els alumnes executen i experimenten

Executen les cel·les d'exemple, modifiquen valors, veuen els errors en temps real i resolen els exercicis proposats.

4

Revisió i feedback

El professor accedeix als quaderns dels alumnes, revisa les solucions i pot deixar comentaris directament a les cel·les. L'entrega és automàtica: no cal enviar cap fitxer.
