📚 llibreria.html

Fitxer de funcions compartides · Projecte GAS Docents

Aquest fitxer conté les funcions JavaScript que injecten la capçalera de navegació i el peu de pàgina a totes les pàgines del projecte. Canviant les dades aquí, el canvi s'aplica a tot arreu.

Com s'inclou a cada pàgina: 

<!DOCTYPE html>
<html lang="ca">
<head>
  <link rel="stylesheet" href="gas-docents.css">
</head>

<!-- Opció A: indicar la pàgina activa com a atribut -->
<body data-pagina="nom_pagina.html">

  <!-- El contingut de la pàgina va aquí -->
  <main> ... </main>

  <!-- Càrrega de la llibreria: injecta nav i footer automàticament -->
  <script src="llibreria.html"></script>

</body>
</html>

Funcions disponibles:

  • iniciarPagina("fitxer.html") — Nav + Footer d'una sola crida
  • injectarNav("fitxer.html") — Només la barra de navegació
  • injectarFooter() — Només el peu de pàgina
// ════════════════════════════════════════════════════════════ function iniciarPagina(pagActiva) { injectarNav(pagActiva); injectarFooter(); } // ════════════════════════════════════════════════════════════ // EXECUCIÓ AUTOMÀTICA per l'Opció A // // Si el té l'atribut data-pagina, la llibreria // s'inicialitza sola en carregar la pàgina: // // // // // Sense necessitat de cridar res manualment. // ════════════════════════════════════════════════════════════ document.addEventListener('DOMContentLoaded', () => { const pagina = document.body.getAttribute('data-pagina'); if (pagina) { iniciarPagina(pagina); } });
